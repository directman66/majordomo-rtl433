Информация
Модуль для работы с USB RTL-SDR тв-тюнерами (стиками) на чипе RTL2832. Модуль работает на linux системах (debian, ubuntu, armbian и др. клонов, основанных на debian). Версия модуля пока pre-alphа. Предварительно должен быть установлен и откомпилирован пакет rtl_433 под вашу систему. Установка на других системах, где нет apt-get также возможна, но команды будут другие.

Установка
Подключаетесь к системе через telnet или ssh 2) выполняете следующие команды: sudo apt-get install make cmake git clone -b feat-rcswitch https://github.com/zuckschwerdt/rtl_433.git git clone https://github.com/zuckschwerdt/rtl_433.git rtl_433_rcswitch cd rtl_433_rcswitch mkdir build cd build cmake .. make
